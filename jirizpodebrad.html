School Project (2019)

Festival visual identity - Jiří z Poděbrad

About

Jiří z Poděbrad was a Czech king born in the 15th century. The assignment was to create a digital project for the 600th anniversary of his birth including visual identity and a website for information about the festival.

Poster

I started by designing a poster. I came up with lots of different versions and at the end picked these top 3.

I decided to go with the pink background version as it turned out to work the best on the web. Matching the styles was by far the biggest challenge of this project.

The website

Following the style, I designed a simple web solution. The website displays the schedule and information about each event and location on a customized Google map.
